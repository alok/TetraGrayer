Technical Details

Doran Coordinates

Horizon-penetrating coordinates for the Kerr metric, regular everywhere outside the ring singularity.

RK4 Integration

Fourth-order Runge-Kutta with adaptive step sizing based on gravitational blueshift.

CSE Optimization

Common Subexpression Elimination in Metal shader codegen for efficient GPU execution.

Tetrad Formalism

Local Lorentz frames for proper handling of photon 4-momentum in curved spacetime.

Getting Started

# Clone the repository
git clone https://github.com/alokbeniwal/TetraGrayer
cd TetraGrayer

# Build with Lake
lake build

# Run all demos
./.lake/build/bin/tetragrayer demos

# GPU benchmark (requires Apple Silicon)
./.lake/build/bin/tetragrayer cpu-gpu

# Metal GPU render
./.lake/build/bin/tetragrayer metal-ffi
