Accretion Disk

Black holes in astrophysical settings are often surrounded by accretion disks - swirling disks of hot gas and dust that emit intense radiation as they fall inward.

TetraGrayer models a thin Keplerian disk in the equatorial plane with: temperature profile following standard thin disk theory, Doppler shift from orbital motion, and gravitational redshift.

The disk lies in the equatorial plane and extends from an inner edge at the ISCO (Innermost Stable Circular Orbit) to an outer edge.

Accretion disk configuration.

structure DiskParams where

Inner edge radius (ISCO)

Outer edge radius

outerRadius : Float := 20.0

Temperature at inner edge (arbitrary units)

innerTemp : Float := 1.0

Rotation direction: 1 = prograde, -1 = retrograde

rotationSign : Float := 1.0 deriving Repr

Temperature Profile

Standard thin disk theory predicts the temperature varies as r to the power of negative three-fourths. The inner disk is hottest (blue-white) and cools toward the outer edge (red).

Temperature at radius r relative to inner edge.

def diskTemperature (r innerR innerTemp : Float) : Float := innerTemp * Float.pow (innerR / r) 0.75

Doppler Shift

The disk rotates at the Keplerian velocity, which is one over the square root of r in geometric units where M equals 1.

Light from the approaching side is blueshifted (hotter-looking), while light from the receding side is redshifted (cooler-looking).

Keplerian orbital velocity at radius r.

def keplerianVelocity (r : Float) : Float := if r > 1.0 then 1.0 / Float.sqrt r else 0.5

Compute Doppler factor from disk motion. Returns greater than 1 for blueshift, less than 1 for redshift.

def dopplerFactor (crossPhi crossR : Float) (rayDirX rayDirY : Float) (rotSign : Float) : Float := let vPhi := keplerianVelocity crossR -- Disk velocity direction (tangent to circle) let diskVelX := -rotSign * vPhi * Float.sin crossPhi let diskVelY := rotSign * vPhi * Float.cos crossPhi -- Normalize ray direction let rayNorm := Float.sqrt (rayDirX * rayDirX + rayDirY * rayDirY) let rdx := if rayNorm > 1e-10 then rayDirX / rayNorm else rayDirX let rdy := if rayNorm > 1e-10 then rayDirY / rayNorm else rayDirY -- Dot product: negative = approaching = blueshift let vDotRay := diskVelX * rdx + diskVelY * rdy 1.0 + 0.8 * (-vDotRay)

Disk Crossing Detection

To render the disk, we check if a ray crosses the equatorial plane (z equals 0) between integration steps. Linear interpolation finds the crossing point.

Check if ray crossed the equatorial plane between steps. Returns (crossed, radius, phi) at crossing.

Color Mapping

Disk color depends on effective temperature (base temp times Doppler factor): Hot regions (T greater than 0.8) appear blue-white, medium regions appear yellow-orange, and cool regions appear red-orange.

RGB color for disk rendering.

structure DiskRGB where r : UInt8 g : UInt8 b : UInt8 deriving Repr

Clamp a value to the 0-1 range.

def clamp01 (x : Float) : Float := max 0.0 (min 1.0 x)
